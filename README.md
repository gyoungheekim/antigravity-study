#COER를 위한 CORTIS 스케줄 타임라인 플랫폼

##이 사이트는 아티스트의 컴백 시즌에 쏟아지는 음악방송, 팬싸인회 등 수많은 스케줄을 팬들이 놓치지 않도록 돕는 스케줄 타임라인 웹사이트입니다.

##🛠️ 사용 방법
1. 상단의 **스케줄 등록**에서 [날짜, 시간, 내용]을 입력하고 원하는 카테고리(`음악방송`, `팬싸인회`, `영통`, `컴백`)를 선택합니다.
2. **'등록하'** 버튼을 누르면 당일 타임라인 리스트에 일정이 추가됩니다.
3. 상단의 **카테고리 탭** 버튼들을 눌러 내가 원하는 종류의 일정만 쏙 골라내어 확인할 수 있습니다.
